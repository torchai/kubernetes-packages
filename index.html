TorchAI Kubernetes Packages

Helm Charts Repository

Welcome to the TorchAI Kubernetes Packages Helm repository. This repository hosts Helm charts for TorchAI's Kubernetes applications.

Available Charts

  • conduwuit - A Kubernetes-native Matrix homeserver

Usage

To use these Helm charts, add this repository to your Helm:

helm repo add torchai https://torchai.github.io/kubernetes-packages
helm repo update

Installing Charts

To install the conduwuit chart:

helm install my-release torchai/conduwuit
